How it Works

01 / DATASET

Training Data

Trained on 2,000 synthetic house records with 8 engineered features. Covers properties from 500-5000 sqft priced $50K to $2M.

02 / MODEL

Random Forest

Uses RandomForestRegressor with 200 trees and max depth 12. Wrapped in a sklearn.Pipeline with StandardScaler preprocessing.

03 / API

REST Endpoint

POST to /predict with a JSON body. Returns predicted_price, range_low, and range_high in USD.

04 / PIPELINE

Deployment

Run train_model.py to generate model.pkl. Start Flask server via python app.py on port 5000.

05 / FEATURES

Input Variables

8 inputs: area_sqft, bedrooms, bathrooms, age_years, garage_spaces, distance_city, has_pool, school_rating.

06 / ACCURACY

Performance

Achieves R2 ~0.93 on the test split. Mean Absolute Error approximately $12,000 on held-out data.
